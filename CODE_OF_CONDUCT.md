# Code of Conduct

## Our Commitment

I'm committed to providing a welcoming and inclusive environment for all individuals engaging with this portfolio and project. This Code of Conduct outlines expectations for respectful interaction.

## Our Standards

### Expected Behavior
- **Be Respectful:** Treat all individuals with respect and dignity
- **Be Constructive:** Provide helpful, actionable feedback
- **Be Inclusive:** Welcome diverse perspectives and backgrounds
- **Be Honest:** Communicate transparently and authentically
- **Be Professional:** Maintain professionalism in all interactions

### Unacceptable Behavior
- Harassment, discrimination, or intimidation
- Offensive language or personal attacks
- Disrespect for others' work or contributions
- Spam or irrelevant content
- Attempts to exploit or manipulate

## Scope

This Code of Conduct applies to:
- GitHub issues and pull requests
- Email communications
- Social media interactions related to this project
- Any representation of this project

## Enforcement

Violations can be reported to: jishnupg2005@gmail.com

I take all reports seriously and will respond appropriately. Confidentiality will be maintained where possible.

## Our Responsibility

As the project owner, I am responsible for:
- Clarifying standards of acceptable behavior
- Taking appropriate action in response to violations
- Maintaining a respectful community environment
- Being fair and consistent in enforcement
